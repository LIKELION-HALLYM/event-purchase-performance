|---|---|---|

| 내 정보 조회 | 이름, 이메일, 전화번호, 기본 배송지 표시 | `index.html`, `app.js` |
| 내 정보 수정 | 수정 버튼, 저장/취소, 화면 즉시 반영 | `app.js` |

| 주문 상세 이동 | 주문 클릭 시 상세 패널 갱신 | `app.js` |
| 찜한 상품 조회 | 찜 상품 리스트 표시 | `app.js` |
| 찜 상품 삭제 | 찜 해제/복구 토글, 카운트 갱신 | `app.js` |
| 최근 본 상품 조회 | 최근 조회 상품 리스트 표시 | `app.js` |

## 검토 결과

- 1차 범위는 마이페이지 기본 기능으로 충분하다.
- 주문 생성, 결제 승인, 장바구니 계산은 각 도메인 기능이므로 마이페이지에서는 조회/이동만 담당한다.
- 백엔드 연동 시 로그인 사용자 ID 기준으로 본인 데이터만 조회하도록 인증/인가 검증이 필요하다.
- 최근 본 상품과 찜 데이터는 AI 추천 개선을 위한 사용자 행동 데이터로 활용 가능하다.

## 2차 후보

- 최근 본 상품 삭제/전체 삭제
- 내 리뷰 조회/수정/삭제
- 배송지 목록 조회/추가/수정/삭제
- 결제 내역 조회
- 비밀번호 변경 이동
- 회원 탈퇴 이동
